# PROJET .

- Paiement de frais de scolarite sur mobile 

## Description

- Application pour les parents qui va permetre de payer les frais de schoolarite d'un eleve a partir de son identifiant scolaire.et de pouvoir suivre les paiements de frais de scolarite


## Fonctionnalites
- Nous voulons juste un MVP pour tester le produit. 


- Authentification : 
	- inscription parent avec (nom , prenom , CIN )
	- login parent.
	
- Gestion de frais de scolarite.
	- Paiement de frais de scoolarite d'un eleve a partir de son Identifiant (le nom de l'eleve s'affiche avec son identifiant).
	- Voir historique de paiements
	
	
# Technologies.
- Flutter et SQFlite pour la persistences des donners.
